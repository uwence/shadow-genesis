## What it does
1.  **Identity Injection**: Installs `IDENTITY.md` with the "Shadow" persona.
2.  **User Context**: Installs `USER.md` with the Master's preferences.
3.  **Memory Structure**: Creates the `memory/` directory with P.A.R.A. subdirectories.
4.  **System Knowledge**: Installs `tacit_knowledge.md` for operational consistency.
5.  **Project Init**: Creates a default project summary.
